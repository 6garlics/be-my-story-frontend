# 토리 (Tori)

## 📗 세상에 없던 나만의 동화책, 토리
토리는 아동이 일기를 작성하면 생성형 AI가 e-동화책으로 만들어주어 친구들과 함께 공유해 볼 수 있는 플랫폼입니다.
<br/><br/>
개발기간: 2023.05.08 ~ now
<br/><br/>


## 🧄 팀 육쪽마늘

## 시작 가이드
### Requirements

For building and running the application you need: 
- node `v20.9.0`
- npm `7.24.2`


### Installation
```
git clone https://github.com/6garlics/tori.git
cd tori
npm install
npm start
```
